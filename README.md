qml.rtmp
========

Easy to use QML plugin, who can read and stream the rtmp stream.

## Depends
* FFMPEG and OpenCV, you need to build it yourself and specify FFMPEG_INCLUDE_PATH, OPENCV_PATH, FFMPEG_LIBRARY_PATH, OPENCV_PATH variables
